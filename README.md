# Plagiarism-Detection
A Python-based plagiarism detection system that analyzes document similarity using NLP techniques and generates detailed plagiarism reports with highlighted matching content.

A Python-based plagiarism detection system that compares documents using Natural Language Processing (NLP) and text similarity algorithms to identify duplicate or highly similar content. The application generates similarity scores, highlights matching text, and provides detailed plagiarism reports through an intuitive interface.

---

## Features

- Compare two or more documents for similarity
- Detect duplicate and paraphrased content
- Calculate plagiarism percentage
- Highlight matching sentences
- Support for TXT, PDF, and DOCX files
- User-friendly interface
- Generate plagiarism reports
- Fast and accurate document comparison

## How It Works

1. Upload one or more documents.
2. Extract text from each document.
3. Preprocess the text using NLP techniques.
4. Convert the text into TF-IDF vectors.
5. Calculate similarity using Cosine Similarity.
6. Display plagiarism percentage and matching content.
7. Generate a plagiarism report.
